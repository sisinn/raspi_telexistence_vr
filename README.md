
# raspi_telexistence_vr  

Display the remote image of Dual Fisheye camera on the VR goggles and experience the telexistence.

---
## Main Equipment  

* Raspberry Pi  
* Insta360 Air  
* BNO035  
* VR Goggles (P-VRG03BK)  
* LCD Monitor (LS055-CGS-LCD-SET)
